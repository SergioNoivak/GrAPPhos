Muitas aplicações em computação necessitam considerar conjunto de conexões entre pares de objetos:
  • Existe um caminho para ir de um objeto a outro seguindo as conexões?
  • Qual é a menor distância entre um objeto e outro objeto?
  • Quantos outros objetos podem ser alcançados a partir de um determinado objeto?
Algoritmos que trabalham com grafos podem resolver esses problemas. Alguns exemplos práticos são:
  • Ajudar máquinas de busca a localizar informações relevantes na Web;
  • Descobrir os melhores casamentos entre posições disponíveis em empresas e pessoas que aplicaram para as posições de interesse;
  • Descobrir qual é a rota mais curta para visitar as principais cidades de uma região.
Exemplos de algoritmos aplicados em grafos são:
  • Algoritmo de Dijkstra;
  • Algoritmo (A-star ou A-estrela);
  • BFS (Breath First Search ou Busca em Largura);
  • DFS (Depth First Search ou Busca em Profundidade)
  • Algoritmo de Ford-Fulkerson;
  • Algoritmo de Bellman-Ford.
Perceba que é bastante aplicado os grafos para a resolução de problemas de busca de rotas.
